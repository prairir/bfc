# BFC

BrainFuck to C transpiler

## build

Typical go build

```console
go build main.go
```

## Usage

```console
Usage of bfc:
  -in-file string
    	input file in BrainFuck (no default)
  -memory-size int
    	memory size of BrainFuck machine (default 30000)
  -out-file string
    	output file (default print to stdout)
```
